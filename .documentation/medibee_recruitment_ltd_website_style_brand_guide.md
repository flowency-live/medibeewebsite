## 3) Colour palette

This palette is **derived directly from the Medibee logo**. It intentionally avoids generic healthcare greens and teals in favour of a calmer, more distinctive slate-and-gold scheme.

### Primary colours
Used for brand identity, key actions, and emphasis.

- **Slate Blue (Primary)**: `#696F8B`
  - Primary brand colour
  - Used for primary CTAs, links, key highlights

- **Deep Slate (Support)**: `#545971`
  - Used for headers, navigation, footer backgrounds, and hover states

### Accent colours
Used sparingly to convey quality and reassurance.

- **Soft Gold**: `#E5D7A2`
  - Used for subtle accents, dividers, and secondary emphasis

- **Rich Gold**: `#D3B25B`
  - Used for highlights, focus states, and limited decorative elements

### Neutral colours
These should dominate the site visually.

- **Mist (Off-White)**: `#E6E3CF`
  - Primary background colour

- **Neutral Grey**: `#9A999B`
  - Used for borders, secondary UI elements, and muted content

### Text colours

- **Primary Text / Ink**: `#09080A`
  - Used for all body copy on light backgrounds

- **Secondary Text**: `#545971`
  - Used for headings, labels, and navigation text

### Functional guidance

- Success states should use **Slate Blue** rather than green
- Warnings and errors should rely on copy and icons first, colour second
- Avoid introducing new semantic colours unless absolutely necessary

## 7) Forms

Forms are a **core trust surface**.

### Form design rules
- Clearly labelled fields (labels always visible)
- Logical grouping of fields
- Single-column layout
- No unnecessary steps

### Field styling
- Soft borders
- Clear focus states
- Error messages in plain English

### Confirmation states
- Clear on-screen confirmation
- Calm success messaging
- No animations or gimmicks

## 8) Imagery & visual elements

### Background usage

- Primary page backgrounds: **Mist** `#E6E3CF`
- Section breaks: White or very subtle tonal shifts only
- Headers / footers: **Deep Slate** `#545971` with light text

### Text contrast rules

- Dark text (`#09080A`) only on light backgrounds
- Light text (`#E6E3CF`) only on Deep Slate backgrounds
- Gold tones must never be used for body text

All text must meet comfortable contrast for long-form reading.

### Photography style (if used)
- Real environments
- Real people
- Natural, neutral lighting
- No heavy colour grading

### Illustrations / icons
- Simple, minimal line icons
- Single-colour (Slate Blue or Ink)
- Used sparingly
